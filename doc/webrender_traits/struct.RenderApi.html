fn set_root_display_list(&self,
                         background_color: Option<ColorF>,
                         epoch: Epoch,
                         viewport_size: LayoutSize,
                         builder: DisplayListBuilder)

Supplies a new frame to WebRender.

Non-blocking, it notifies a worker process which processes the display list. When it's done and a RenderNotifier has been set in webrender::renderer::Renderer, new_frame_ready() gets called.

Note: Scrolling doesn't require an own Frame.

Arguments:

  • background_color: The background color of this pipeline.
  • epoch: The unique Frame ID, monotonically increasing.
  • pipeline_id: The ID of the pipeline that is supplying this display list.
  • viewport_size: The size of the viewport for this frame.
  • display_list: The root Display list used in this frame.
  • auxiliary_lists: Various items that the display lists and stacking contexts reference.
