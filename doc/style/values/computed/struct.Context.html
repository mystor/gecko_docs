A Context is all the data a specified value could ever need to compute itself and be transformed to a computed value.

Fields

is_root_element: bool

Whether the current element is the root element.

viewport_size: Size2D<Au>

The style we're inheriting from.

style: ComputedValues

Values access through this need to be in the properties "computed early": color, text-decoration, font-size, display, position, float, border-*-style, outline-style, font-family, writing-mode...

font_metrics_provider: Option<&'a FontMetricsProvider>

A font metrics provider, used to access font metrics to implement font-relative units.
