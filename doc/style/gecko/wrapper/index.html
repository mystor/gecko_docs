style::gecko

Module style::gecko::wrapper [−] [src]

Wrapper definitions on top of Gecko types in order to be used in the style system.

This really follows the Servo pattern in components/script/layout_wrapper.rs.

This theoretically should live in its own crate, but now it lives in the style system it's kind of pointless in the Stylo case, and only Servo forces the separation between the style system implementation and everything else.
